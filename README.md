# 🌐 LingoPro — Language Translation Tool

A modern, responsive web-based language translator that converts text between multiple languages including English, Spanish, French, German, Chinese, Urdu, Arabic, and Japanese — complete with native script rendering, voice input, and text-to-speech.

> 🎓 This project was built as **Task 1** of my Web Development Internship at **CodeAlpha**.


## ✨ Features

- 🔤 Translate text between 8+ languages
- 🔄 Swap source and target languages instantly
- 🎙️ Voice input — speak instead of typing
- 🔊 Text-to-speech — listen to the translated text
- 📋 One-click copy to clipboard
- 📁 Upload `.txt` files for translation
- 🕘 Translation history (saved locally)
- ⭐ Save favorite translations
- 🌙 Light/Dark mode toggle
- 📱 Fully responsive — works on mobile and desktop
- 🈶 Proper native script rendering for Arabic, Urdu, and Japanese (right-to-left support included)

## 🛠️ Built With

- **HTML5** — structure
- **CSS3** — styling, animations, responsive design
- **JavaScript (Vanilla)** — logic and API integration
- **Google Translate API (public endpoint)** — translation engine
- **Web Speech API** — voice input and text-to-speech
- **Font Awesome** — icons
- **Google Fonts** — Plus Jakarta Sans, Noto Sans Arabic, Noto Nastaliq Urdu, Noto Sans JP

## 🚀 How to Run

1. Clone this repository
2. Open the folder
3. Double-click `index.html` (or open it with the **Live Server** extension in VS Code)
4. Start translating!

No installation, no API key, no backend server required — it runs entirely in the browser.

## 🎯 How It Works

1. User types or speaks text into the input box
2. User selects the source and target languages
3. On clicking "Translate," the app sends a request to the translation API
4. The translated text is displayed in the output box, in the correct script and direction (left-to-right or right-to-left)
5. User can copy, listen to, or save the translation
